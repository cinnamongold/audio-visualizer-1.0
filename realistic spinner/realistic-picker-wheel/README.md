Ultrarealistic 3D Reflective Picker Wheel (Browser Demo)
This is a self-contained browser demo that renders a 3D picker wheel with a glossy, reflective surface and basic inertial physics. It uses Three.js loaded from a CDN and draws segment labels on a canvas texture for a realistic look.

How to run:
- Open realistic-picker-wheel/index.html in a modern browser.

What you get:
- A thin 3D disc that spins around the vertical axis.
- Environment reflections via a simple cube-map environment map.
- Inertia on release for a smooth spin.
- A simple UI ring indicator to show the selected segment.

Notes:
- This is a minimal, self-contained example. For production, you may want to modularize and optimize texture updates, add ARIA controls, and fine-tune lighting/shadows.
